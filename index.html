Do you need a degree to be a web developer?

Today I have been exploring the best way to become a web developer, I have read so much on other peoples blogs, searched on reddit and I have come to the conclusion that no you don't need a degree to become a web developer.

Although you dont need a degree it is helpfull to have knowledge on coding, you will need knowldege on:

  • Html
  • Css
  • Javascript
It is essential to learn code to get a job in web development, you may also want to get yourself to a place where you can code a website like this to showcase your skills. although these code languages are the very basic standard you can learn even more languages like C# (Pronounced C Sharp). You may find it difficult to get a job without proof of skills and without relivent experience.

To get relivent experience you may want to:

  • Find an internship
  • Use your skills to develope websites FREELANCE
  • Offering to help companies in exchange for resume recomendations
It's also good to keep in mind that places you get an internship or offer help may land you a job with that company afterwards, also if youve been doing freelance work you may want to stick with that, if its going well for you.
